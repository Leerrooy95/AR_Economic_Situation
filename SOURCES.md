## How to Verify Specific Data Points

### Violent Crime Rates
1. Visit FBI Crime Data Explorer: https://cde.ucr.cjis.gov/
2. Select "Crime Data API" or download datasets
3. Filter by agency (West Memphis PD, Southaven PD)
4. Calculate rate per 100,000 using population

### Education Proficiency
1. Visit respective state education portal
2. Search for district (West Memphis, Southaven, Texarkana)
3. Navigate to state assessment results
4. Compare percentage meeting proficiency standards

### Sales Tax Rates
1. Arkansas: https://www.salestaxhandbook.com/arkansas/rates
2. Mississippi: https://www.salestaxhandbook.com/mississippi/rates  
3. Texas: https://www.salestaxhandbook.com/texas/rates
4. Cross-reference with official state revenue department

### Population and Migration
1. Census QuickFacts: https://www.census.gov/quickfacts/
2. Enter county name
3. Compare "Population estimates" and "Population % change"

## Data Access Notes
